# IPL 2025 Auction Dashboard 🏏

An interactive Power BI dashboard analyzing IPL 2025 auction data.

## Pages
- **Auction Overview** — Total spend, team-wise spending, role breakdown
- **Player Analysis** — Most expensive players, full player table
- **Team Budget** — Team-specific stats and squad details

## Tools Used
- Power BI, DAX, Power Query

## Dataset
- Source: Kaggle (IPL 2025 Auction Players)
- File: `ipl_2025_auction_players.csv`

## How to Open
- DOwnload the dataset first.
- Download the `.pbix` file and open with Power BI Desktop (free).
